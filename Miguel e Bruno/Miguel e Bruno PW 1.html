Quem foi Timothy Berners-lee? e qual sua importancia para o html?

é um físico britânico, cientista da computação e professor do MIT. A sua importância no HTML foi de que
ele acabou tornando o HTML como linguagem padrão.

qual necessidade influenciou a criação o HTML?

Em uma época em que a Internet mal engatinhava, um grupo de pesquisadores britânicos viram a necessidade de uma
ferramenta capaz de acrescentar elementos complementares a textos, como títulos, citações, imagens, linkar outro
artigos, entre outras ferramentas.


Explique o papel do W3C para o desenvolvimento do HTML

O papel do W3C é gerenciar os padrões da Web e para isso propõe um fluxo de aprovação destes padrões.

A evolução do HTML

O HTML venceu a disputa entre as linguagens demarcação web por vários motivos,
entre eles: simplicidade, flexibilidade, portabilidade e evolução.
